Quick Start

bash 
# Clone the repository
git clone https://github.com/shyamalchandra/MRFcompiler.git
cd MRFcompiler

# Build the compiler
make

# Run with example
./mrf_compiler example.txt output.qasm

# Export to Qiskit
./mrf_compiler -f qiskit example.txt circuit.py
