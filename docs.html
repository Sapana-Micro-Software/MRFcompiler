API Reference

Command Line Interface

./mrf_compiler [options] [input_file] [output_file]

Options:
  -f, --framework <name>  Output framework
  -a, --all               Export to all frameworks
  -h, --help              Show help message

Input Format

TYPE directed|undirected
NODE <id> <name> [num_states]
EDGE <from> <to> [directed]

Supported Frameworks

  • qasm - OpenQASM 2.0
  • qiskit - IBM Qiskit
  • cirq - Google Cirq
  • pennylane - Xanadu PennyLane
  • qsharp - Microsoft Q#
  • braket - AWS Braket
  • qulacs - Qulacs
  • tfq - TensorFlow Quantum
