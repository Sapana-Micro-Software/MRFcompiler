Examples

Learn by example with practical MRF compiler usage

Basic Example: Undirected Graph

Convert a simple undirected graphical model to Qiskit circuit.

Input File (example.txt)

text 
TYPE undirected
NODE 0 A 2
NODE 1 B 2
NODE 2 C 2
EDGE 0 1
EDGE 1 2

Command

bash 
./mrf_compiler -f qiskit example.txt circuit.py

Generated Output

python 
from qiskit import QuantumCircuit, QuantumRegister, ClassicalRegister

def create_mrf_circuit():
    qr = QuantumRegister(3, 'q')
    cr = ClassicalRegister(3, 'c')
    qc = QuantumCircuit(qr, cr)
    
    qc.h(0)
    qc.h(1)
    qc.h(2)
    # ... additional gates ...
    
    return qc

Export to All Frameworks

Generate circuit code for all supported frameworks at once.

bash 
./mrf_compiler -a example.txt

This generates:

  • output_qasm.qasm - OpenQASM format
  • output_qiskit.py - Qiskit Python code
  • output_cirq.py - Cirq Python code
  • output_pennylane.py - PennyLane Python code
  • output_qsharp.qs - Q# source code
  • output_braket.py - AWS Braket Python code
  • output_qulacs.py - Qulacs Python code
  • output_tfq.py - TensorFlow Quantum code

Directed Graph Example

Convert a directed Bayesian network to MRF and then to quantum circuit.

Input File (bayesian.txt)

text 
TYPE directed
NODE 0 Rain 2
NODE 1 Sprinkler 2
NODE 2 WetGrass 2
EDGE 0 2 directed
EDGE 1 2 directed

Command

bash 
./mrf_compiler -f cirq bayesian.txt bayesian_circuit.py
